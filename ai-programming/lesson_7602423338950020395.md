**首先教大家如何不花一分钱接入Claude模型。**

这里我们要进入一个Google推出的工具Antigravity，中文名是反重力，它本身是一个AI编程的工具，内置了Gemini和Claude这两个顶级的模型。在给它点赞上找到Antigravity这个插件，复制安装指令，回到OpenCode里粘贴，OpenCode就会自动下载安装。安装完成后重新打开终端，复制这条指令回车，然后我们选择Google，选择带有Antigravity字样的这行，回车，接着就会跳转到浏览器，登入自己的Google账号，复制这个流量粘贴到终端回车，这样我们就配置完成了。重新打开一个终端，进入OpenCode，输入/model，我们就能看到Gemini 3 Pro和Claude 4.5 Opus这两个模型了。我们测试一下，输入"介绍一下你自己"，可以看到它正常输出，证明我们接入成功了。从现在开始，全是顶级大脑在为你打工。

**有了顶级大脑，还得有强力的四肢。接下来我们要给OpenCode装上它最火的插件包：Omni OpenCode。**

它内置了7个顶级的编程Agent，通过多智能体协作帮你自动完成写代码、调Bug、搭建项目结构这些开发工作，都不用手动处理很多繁琐的细节。据说开发者花了2.4万美金才搞出来的。安装也很简单，复制给它点赞上的安装指令丢给OpenCode，它会询问你是否有Claude的GPT和Gemini的会员，你就如实回答，大概需要一段时间安装好了。重启OpenCode，可以使用/item命令从7大智能体中挑选一个帮你执行任务，或者直接在提示词里输入"ultrawork"，就可以使用Omni OpenCode。

**接下来教大家如何在OpenCode使用Skills。**

OpenCode可以无缝衔接Claude Code创建的Skills。首先来到根目录，新建一个文件夹命名为第二个OpenCode，再新建一个Skills的子文件夹，然后把你的Skills复制到这个文件夹，就可以在OpenCode使用了。当然我们也可以直接在OpenCode创建Skills，需要先安装Skills Creator。有了它就可以用自然语言来创建Skills。我们让OpenCode自己去安装这个Skills Creator，安装完成后我们测试一下，让它创建一个翻译功能的Skills，它会根据你的要求自动执行并创建，创建成功后重启OpenCode就可以看到这个Skills已经有了。

**最后是配置MCP。**

我一般都是让它自己来安装MCP。比如安装一个浏览器MCP，我会这样在OpenCode输入"安装Chrome MCP Server"，稍等一会它就安装成功了。有些MCP需要配置Key，我们也可以把Key给它，让它自己去配置，这样我们就不需要自己手动去配置了。然后输入/mcps就能看到所有MCP。

从白嫖顶级模型，到手握专属Skills，再到部署全能MCP，现在的OpenCode应该已经是你的超级编程助手。OpenCode不仅能进行零代码编程，更能帮你处理翻译、整理资料、智能写作。下一期我会带大家继续进行进阶的学习，大家记得关注，好跟着阿亮学AI。

### 2.3 核心概念精讲

#### Antigravity（免费接入顶级模型）

**是什么**：Google推出的AI编程工具，内置Gemini和Claude两个顶级模型，通过插件形式接入OpenCode，实现免费使用。

**为什么重要**：Claude和Gemini是目前最强的编程模型，通过Antigravity可以不花一分钱直接在OpenCode中使用。

**详细解释**：
配置流程：
1. 在给它点赞平台找到Antigravity插件
2. 复制安装指令 → 粘贴到OpenCode → 自动安装
3. 重开终端 → 选择Google → 选Antigravity选项
4. 跳转浏览器登录Google账号 → 复制凭证粘贴回终端
5. 输入`/model` 验证Gemini 3 Pro和Claude 4.5 Opus是否出现

**实际应用场景**：零成本体验顶级AI编程模型，适合学习阶段或预算有限的开发者。

---

#### Omni OpenCode（多智能体编程插件）

**是什么**：OpenCode的超级增强插件，内置7个专业编程Agent，通过多智能体协作自动完成复杂编程任务。

**为什么重要**：单一AI模型处理复杂项目效率有限，多智能体协作能分工处理不同任务，提升整体效率。

**详细解释**：
| 触发方式 | 说明 |
|---------|------|
| `/item` 命令 | 从7个Agent中选择一个执行特定任务 |
| `ultrawork` 关键词 | 直接在提示词中触发Omni模式 |

**实际应用场景**：复杂项目开发、自动调Bug、搭建项目结构。

---

#### Skills在OpenCode中的使用

**是什么**：OpenCode与Claude Code共享Skills生态，可以直接复用Claude Code创建的Skills，也可以在OpenCode内创建新Skills。

**为什么重要**：Skills是AI工具的核心竞争力，能将工作流标准化封装，实现跨工具复用。

**详细解释**：
- **方式一**：根目录新建文件夹 → 子文件夹Skills → 复制Skills文件进去
- **方式二**：安装Skills Creator → 自然语言描述需求 → 自动创建Skills
- 创建完成后重启OpenCode即可使用

**实际应用场景**：将Claude Code中已有的写作/翻译/代码审查Skills迁移到OpenCode使用。
