### 2.3 核心概念精讲

#### 概念一：全局汉化（深层设置）

**是什么**：对 Claude Code 进行深层配置，让所有命令界面和提示信息以中文显示。

**为什么重要**：英文命令是大多数国内用户上手 Claude Code 的最大障碍，汉化后可以大幅降低认知负担。

**详细解释**：通过修改 Claude Code 的设置，将命令提示、菜单选项等全部切换为中文大白话，不需要再额外开翻译工具。

**实际应用场景**：日常使用 Claude Code 执行各类编程任务时，所有交互界面均以中文呈现。

---

#### 概念二：运行时中文说明书

**是什么**：在 Claude Code 执行每条命令时，在命令下方自动弹出粉红色中文提示框，解释该命令的作用和原因。

**为什么重要**：让用户不再对"黑箱"感到恐惧，每一步操作都透明可读，边用边学。

**详细解释**：
- 弹出位置：命令运行区域下方
- 显示内容：① 当前命令在做什么；② 为什么需要执行这个命令
- 显示样式：粉红色中文提示（视觉上易于区分）

**实际应用场景**：初学者跑第一个项目时，可以实时理解每一步命令的含义，不再懵圈。
