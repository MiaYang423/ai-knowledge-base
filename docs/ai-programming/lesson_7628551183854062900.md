**是什么**：通过输入一句自然语言描述（如"画一张 AWS 的架构图，玻璃态风格"），工具自动理解需求并生成对应架构图。

**为什么重要**：
传统画架构图需要手动拖拽组件、对齐布局、调整配色，耗时且门槛高；对话式生成将这一过程压缩为"说一句话"，极大降低了出图成本。

**详细解释**：

| 传统方式 | Skills 对话方式 |
|--------|--------------|
| 手动选择图形组件 | 自然语言描述需求 |
| 手动布局对齐 | 自动识别并排版 |
| 手动调整配色 | 自动配色 |
| 输出质量依赖操作者经验 | 直接输出高清图片 |

**实际应用场景**：
快速为技术方案文档生成 AWS 部署架构图，无需打开 Visio 或 Lucidchart。
