### 2.3 核心概念精讲

#### 概念一：AI 配图 Skills

**是什么**：一个托管在 GitHub 上的 AI 技能模块，专门用于为文章自动生成配套图片。

**为什么重要**：
内容创作中，配图往往是最耗时的环节之一，这个工具将配图流程自动化，显著降低创作门槛。

**详细解释**：
- 支持生成封面图、信息图、文章插图三类图片
- 在 GitHub 上已积累 14K Star，说明社区认可度较高
- 内置了文章插图的最佳实践技巧

**实际应用场景**：
写完一篇小红书图文后，直接调用该 Skills 批量生成封面和配图，无需手动设计。

---

#### 概念二：风格 / 布局 / 密度的自定义

**是什么**：用户在生成图片时可以指定的三个核心参数，控制最终图片的视觉呈现效果。

**为什么重要**：
不同平台、不同内容类型对图片风格要求不同，可自定义意味着工具有更强的适配性，不是"千篇一律"的输出。

**详细解释**：

| 参数 | 含义 |
|------|------|
| 风格 | 图片的整体视觉风格，类型较丰富（如卡通、扁平、写实等） |
| 布局 | 图片内元素的排列方式 |
| 密度 | 图片中信息或视觉元素的疏密程度 |

**实际应用场景**：
做知识类内容时选择高密度信息图；做生活类内容时选择简洁卡通风格封面。

---

#### 概念三：图文生产流水线

**是什么**：将"写文章→配插图→生成封面"整合成一套连贯的自动化工作流。

**为什么重要**：
单点工具只能解决单一问题，而流水线思维能让整个创作过程系统化、可复用，真正提升效率。

**详细解释**：
- 该 Skills 不只是生成单张图，而是串联了文章插图的完整生产逻辑
- 内置插图技巧，降低用户自己调参的学习成本

**实际应用场景**：
创作者只需输入文章内容，工具自动按流水线输出一套完整的配图方案。
