## 一、视频原文

如果我重新来过，重新开始学AI，那么我一定会做这三件事。其实之前我在星球内都有反复说过，那本期视频我来完整表述一遍。

**第一，使用Claude Code。即使没用两天就被封号，也要用。**

今天我们已经进入到Agentic AI阶段，在这个阶段目前最有代表性的工具就是Claude Code。它跟之前的一些聊天机器人是完全不一样的，完全是天壤之别。聊天机器人输出的是文字和图片，这个很好理解。而Claude Code这种工具通过代码创造更多的东西，通过命令去控制更多的东西。这个关卡别人的描述，或者你看了几个视频，是根本想象不出来的。所以如果我今天要重新来过的话，第一件事一定是亲自上手使用Claude Code，只有上手了才有真正的体感。

AI正在飞速发展，这话听起来确实特别烂大街，但是我真的找不出比"飞速发展"更准确的词来描述了。在这种情况下，一定不能做井底之蛙，比如觉得AI不过如此。一定要亲自探出脑袋去看一看，亲眼看了才知道天究竟有多大。这个就是我前面说即使被封号了也要用的原因。不管之后你是否还继续使用Claude Code，反正在最初一定要体验，一定要摸一摸。

多说两句，之前我出视频介绍过Google Stitch，当时我就判断接下来要失业的很可能是设计师。很多人不服气，觉得我就是在传播焦虑。结果前两天Claude Code站出来了。你以为设计就是天马行空，就是所谓的感觉？你就看着吧，从最标准化的工作开始，AI会一步一步吃掉设计师的份额。那扯远了，咱们继续。

**第二，彻底解决基础设施的问题。**

最先进的AI在国外，国内大概落后一年，这个就是事实，跟爱不爱国、是否支持国产AI没有关系。由于众所周知的原因，要稳定使用国外的AI真的需要一套基础设施，这个也是我们在星球里老生常谈的话题，我不断地被问到，我也不断地强调。所以如果我今天重新来过，我一定会专门花时间去优先解决，否则刚才说的那些AI我也用不了。

比如支付的问题，最简单的做法是办一张国内的Visa或者Mastercard信用卡，说是国内的卡，但还是挺有用的。比如我一直用招商的Mastercard订阅Gemini，用建行的Visa订阅Copilot。至于那些没法用国内卡订阅的AI，还可以通过美区iOS账户加礼品卡的方式来完成支付。那更进一步，还可以办一张Capital One的实体信用卡，这个是国外的卡，可能是因为我申请了ITIN，所以办这些都非常顺利。

说实话，我看到最多的情况就是因为基础设施的原因导致没法使用最先进的AI，不知道是懒还是什么原因，反正就是不愿意彻底解决，然后一次又一次又一次地被卡住，我真的都无法理解了。

**第三，资料Markdown化，能力Skill化。**

所有人都在说要学AI，那么要学到什么程度呢？你总得有一个衡量标准吧。我之前出过关于学习AI的教程，之后又在星球内特意发了一个帖子，就是关于标准的。我认为在现阶段，衡量标准就有一个：个体和AI能否成为认知共同体。

一方面，个体清楚知道AI的边界，这个边界包括AI能力和支持的界限、不同模型工具的差异等等；另一方面，AI能够基于对个体的了解进行动态的预判。那要做到这一点，你得让AI了解你。

所以如果我今天重新来过，我一定会把我所有沉淀下来的资料都变成Markdown格式，因为这个格式AI能读懂，咱们人类也能看得懂。至于什么Word、PDF、PPT，我肯定是不会再搞了，那些都太复杂了，AI看不懂。既然AI看不懂，那这些格式一定会越来越被边缘化。

另外我还会把我所有的能力都Skill化。说实话，这个过程挺有难度的，能把脑子里的东西变成Skill，变成明明白白的SOP，变成AI也能理解的文字，这其实就是一个自我提升的过程。这也是我今年一直在星球内建议大家做三个Skill的原因，不仅可以赋能自己的AI系统，还能充分梳理自己的工作流。

以上这三条建议都是我过去这两三年的经验总结，也是我在星球内反复强调的。至于具体怎么做，你可以去搜索，可以去问AI，别来问我。如果你搞不定的话，那就别想着学AI了。OK，以上就是本期内容，想了解AI，想成为超级个体，想找到志同道合的人，就来我们Newtype社群，咱们下期见。

### 2.3 核心概念精讲

#### 概念一：Agentic AI 与 Claude Code

**是什么**：Agentic AI 是当前AI发展的新阶段，代表工具不再只是对话，而是能通过代码和命令主动创造、控制更多事物。Claude Code 是这一阶段目前最有代表性的工具。

**为什么重要**：
这标志着AI从"回答问题"进化到"执行任务"，能力量级完全不同。作者认为，光靠别人描述或看视频根本无法想象这种差距，必须亲自上手才能建立真实认知。

**详细解释**：

| 类型 | 代表工具 | 输出形式 | 能力边界 |
|------|----------|----------|----------|
| 聊天机器人 | ChatGPT、Gemini 等 | 文字、图片 | 信息生成 |
| Agentic AI | Claude Code | 代码、操作、控制 | 创造和控制实际事物 |

**实际应用场景**：
作者提到 Google Stitch 和 Claude Code 正在侵入设计师的工作领域，从最标准化的设计工作开始逐步替代，这是 Agentic AI 能力落地的真实案例。

---

#### 概念二：AI 学习的基础设施

**是什么**：能够稳定访问和付费订阅国外主流AI工具所需的一整套技术和支付条件。

**为什么重要**：
作者直接点出：最先进的AI在国外，国内落后约一年。基础设施不解决，所有学习计划都是空谈。这是最常见、也最容易被拖延的障碍。

**详细解释**：

| 层级 | 方案 | 适用场景 |
|------|------|----------|
| 基础层 | 国内 Visa / Mastercard（招商 Mastercard、建行 Visa 等） | 订阅 Gemini、Copilot 等 |
| 进阶层 | 美区 iOS 账户 + 礼品卡 | 国内卡无法订阅的AI服务 |
| 高阶层 | 国外实体信用卡（如 Capital One） | 更广泛的国外服务订阅 |

**实际应用场景**：
作者自己用招商 Mastercard 订阅 Gemini，用建行 Visa 订阅 Copilot，并通过申请 ITIN 顺利办理了 Capital One 信用卡。

---

#### 概念三：个体与AI成为"认知共同体"

**是什么**：作者提出的衡量AI学习程度的核心标准——个体和AI之间建立起深度的双向了解，能够协同思考和工作。

**为什么重要**：
"学AI"这件事没有终点，需要一个可衡量的阶段性标准。这个标准把目标从模糊的"会用AI"变成了具体的"AI能预判我的需求"。

**详细解释**：

认知共同体需要满足两个条件：

- **个体侧**：清楚知道AI的能力边界，包括不同模型的差异、工具的适用范围和局限
- **AI侧**：基于对个体的充分了解，能动态预判个体的需求和偏好

要实现这一点，核心动作是**让AI真正了解你**，也就是下面的第四个概念。

**实际应用场景**：
这是作者在星球内反复强调的学习衡量标准，也是他建议大家整理资料和Skill的根本原因。

---

#### 概念四：资料 Markdown 化 + 能力 Skill 化

**是什么**：
- **Markdown 化**：把个人积累的所有资料转换为 Markdown 格式，让AI能高效读取和理解
- **Skill 化**：把个人的隐性能力整理成明确的 SOP（标准操作流程），变成AI也能理解的结构化文字

**为什么重要**：
AI无法理解复杂的 Word、PDF、PPT 格式，这些格式会越来越被边缘化。只有结构化、AI可读的内容，才能真正让AI成为你的认知伙伴。

**详细解释**：

| 动作 | 原格式 | 目标格式 | 核心价值 |
|------|--------|----------|----------|
| 资料 Markdown 化 | Word / PDF / PPT | Markdown (.md) | AI可读，人也可读 |
| 能力 Skill 化 | 脑子里的隐性经验 | 明确的 SOP 文档 | 赋能AI + 梳理自身工作流 |

**实际应用场景**：
作者在星球内建议大家今年完成三个 Skill 的整理，这个过程本身也是一次深度的自我梳理和提升。
