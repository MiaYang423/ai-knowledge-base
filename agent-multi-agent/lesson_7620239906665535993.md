### 2.3 核心概念精讲

#### Follow Builders not Influencers（Skill）

**是什么**：一个可安装到 Agent 的技能模块，专门追踪 25 位精选 AI 实践者（Builders）在推特上的关键动态和观点，并定期汇总推送。

**为什么重要**：AI 领域信息量巨大，但真正有价值的内容来自少数真正在做事的人（Builders），而非流量型博主（Influencers）；通过 Skill 自动过滤，大幅提升信息获取效率。

**详细解释**：
- 信息源：25 位精选 AI Builders 的推特账号
- 推送频率：可设置每天或每周
- 触发方式：Agent 定时自动执行（如早上 10 点）
- 输出内容：关键动态、观点摘要

**实际应用场景**：每天早上 10 点让 Agent 自动抓取并推送，替代手动刷推特，用 10 分钟完成当天 AI 信息的核心摄入。

---

#### Agent Skill（技能模块）

**是什么**：可安装到 Agent 上的功能扩展模块，让 Agent 具备特定的自动化能力，无需手动编写代码。

**为什么重要**：Skill 是 Agent 能力的积木，通过组合不同 Skill 可以快速搭建个性化的自动化工作流，降低使用门槛。

**详细解释**：
- 安装方式：直接在 Agent 界面安装对应 Skill
- 使用方式：安装后即可通过自然语言指令触发
- 扩展性：不同场景可安装不同 Skill

**实际应用场景**：将信息订阅、数据分析、内容生成等能力以 Skill 形式模块化，按需组合到个人 Agent 工作流中。
